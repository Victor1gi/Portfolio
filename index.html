Pong: Desafie a Gravidade

Este projeto é um jogo desenvolvido com p5.js onde o jogador controla uma raquete que precisa impedir que uma bolinha caia. A dificuldade aumenta conforme o tempo passa. Ideal para praticar lógica, colisão e manipulação de objetos com JavaScript.

Usei conceitos como velocidade, direção e detecção de colisão para tornar o jogo mais desafiador. Um ótimo exemplo de como a programação pode criar jogos interativos com poucas linhas de código.
